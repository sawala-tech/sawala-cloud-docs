**Kontena** is Sawala Cloud's _headless CMS_ — a Content Management System with no built-in front-end. Editors define _schemas_ (content shapes, e.g. a `post` with `title`, `body`, `coverImage`) and fill in _entries_ against them. Each entry is readable through a public REST API from anywhere.

## Core concepts

- **Schema** — content shape.
- **Collection** — many entries per schema.
- **Single** — one entry per locale per schema.
- **Public API key** — read-only token for client-side use.

## Field types

Kontena supports 14 field types: `text`, `richtext`, `markdown`, `number`, `boolean`, `date`, `select`, `multiselect`, `relation`, `media`, `json`, `repeater`, `blocks`, and `component`.
