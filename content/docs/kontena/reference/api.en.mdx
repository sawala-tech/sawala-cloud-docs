## Base URL

```
https://api.sawala.cloud/public/kontena
```

## Required header

```
X-API-Key: pk_live_xxx
```

A public key (`pk_live_…`) is enough for every read endpoint here. See [how to mint an API key](/en/docs/kontena/how-to-mint-api-key).

## Endpoints

### Collection (multiple entries)

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/projects/{projId}/content/collection/{schemaSlug}` | List entries in a collection |
| `GET` | `/projects/{projId}/content/collection/{schemaSlug}/{id}` | Fetch one entry by ID |

### Single (one entry per locale)

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/projects/{projId}/content/single/{schemaSlug}` | Fetch the singleton entry |

### Schemas

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/projects/{projId}/schemas` | List all schemas |
| `GET` | `/projects/{projId}/schemas/{schemaId}` | Fetch a schema definition |

## Query parameters

| Param | Values | Description |
| --- | --- | --- |
| `locale` | `id`, `en`, … | Filter by entry locale |
| `q` | string | Substring filter on `title` and `slug` |
| `publicationState` | `live` (default), `preview` | `preview` requires a secret key |
| `limit` | integer (default 20, max 100) | Entries per page |
| `cursor` | string | Cursor for the next page |

## Error codes

| HTTP | Body | Cause |
| --- | --- | --- |
| `400` | `{ "error": "..." }` | Invalid params/body |
| `401` | `{ "error": "unauthorized" }` | Missing/wrong API key |
| `403` | `{ "error": "forbidden" }` | Token has no access to that project |
| `404` | `{ "error": "not found" }` | Entry / schema / project doesn't exist |
| `429` | `{ "error": "rate limit exceeded" }` | Too many requests |

## Rate limit

Default: 100 requests per minute per API key. Contact Sawala to raise this.
