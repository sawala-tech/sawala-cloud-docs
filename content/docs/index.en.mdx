These docs are built with [Fumadocs](https://fumadocs.dev), Next.js 16, and Tailwind v4 — the same stack the rest of Sawala's products run on.

## Structure

Content follows the [Diátaxis](https://diataxis.fr/) shape: every product has a **Tutorial** (to learn), **How-to Guides** (to get things done), **Reference** (to look things up), and **Explanation** (to understand the concepts).

Content is still being filled in. Per-product pages will land in the next milestone (M6 in [`PLAN-public-docs-framework.md`](https://github.com/sawala-tech/sawala-cloud/blob/main/docs/plan/PLAN-public-docs-framework.md)).
