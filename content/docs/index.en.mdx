Sawala Cloud is a suite of building blocks for shipping web products — content, files, forms, hosting, data, and more — each usable on its own with a single API key. New to the platform? Start with [General Concepts](/docs/general-concepts/what-is-sawala-cloud).

## Products

| Product | What it does |
|---|---|
| [Kontena](/docs/kontena/what-is-kontena) | Headless CMS — model content and serve it over an API. |
| [Berkasna](/docs/berkasna/what-is-berkasna) | File & media management — images, video, documents, ready to use from your app. |
| [Formulir](/docs/formulir/what-is-formulir) | No-code form builder — create, embed, and collect submissions. |
| [Kodena](/docs/kodena/what-is-kodena) | Hosting & deployment — ship a static site or Worker to the edge with one command. |
| [Datana](/docs/datana/what-is-datana) | Structured-data platform — typed collections, records, and server-side queries, private by default. |
| [Sebar](/docs/sebar/what-is-sebar) | Transactional email — send templated mail through one API and track delivery. |
| [Akuna](/docs/akuna/what-is-akuna) | Membership — bring your own Clerk app and add sign-in, a member directory, and a drop-in embed. |
| [Tugasna](/docs/tugasna/what-is-tugasna) | Work tracking — boards, columns, and a backlog of rich items. |
| [Kiosna](/docs/kiosna/what-is-kiosna) | B2B billing — catalog, contracts, invoices, payments, and renewals. |
| [Kelasna](/docs/kelasna/what-is-kelasna) | Nano-learning marketplace — author short courses; learners enrol, complete, and earn certificates. |
| [Ajena](/docs/ajena/what-is-ajena) | AI chat agent — embed an assistant that answers from a knowledge base you control. |

## How these docs are organized

Each product follows the [Diátaxis](https://diataxis.fr/) shape — a **Tutorial** to learn, **How-to Guides** to get things done, **Reference** to look things up, and **Explanation** to understand the concepts. Coverage is filling in product by product; if a section isn't there yet, it's on the way.
