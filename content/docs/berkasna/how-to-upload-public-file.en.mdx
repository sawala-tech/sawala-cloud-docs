When you need to accept a user upload, store it in Berkasna, and immediately get back a public URL — use the three-call pattern below.

## Steps

1. **Request an upload slot** from your app (server-side, secret key):

   ```bash
   POST https://api.sawala.cloud/public/berkasna/upload/presigned
   X-API-Key: sk_live_xxx
   ```

2. **PUT the file to `uploadUrl`** from the browser.

3. **Mark complete** from your app server.
