**Formulir** is Sawala Cloud's form builder. You define fields (text, email, dropdown, file upload, and more) in the dashboard, then embed the form on your site or app — either with a one-line iframe or a React component. Submissions are stored on Sawala and accessible from the dashboard or via API.

## Core concepts

- **Form** — name, slug, list of fields, settings (e.g. redirect URL after submit).
- **Field** — one input. Supported types: text, email, textarea, number, checkbox, radio, select, file upload.
- **Submission** — one filled instance of the form. Stored with timestamp + metadata.
- **Embed** — how the form renders on your site: iframe or React component.
